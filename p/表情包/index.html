Guide to emoji usage in Hugo

Jul 20, 2024
阅读时长: 1 分钟

表情符号可以通过多种方式在Hugo项目中启用。 <！-更多- > ’ emojify ‘函数可以在模板中直接调用，也可以在Inline Shortcodes中调用。

要全局启用emoji，在你的网站的配置中设置’ enableEmoji ‘为’ true ‘，然后你可以直接在内容文件中键入emoji速记代码;如。

< p > < span class = " nowrap”}> <跨类= >“emojify🙈跨度< / > <代码>:see_no_evil: < /代码> < / span > <跨类=“nowrap”}> <跨类= >“emojify🙉跨度< / > <代码>:hear_no_evil: < /代码> < / span > <跨类=“nowrap”}> <跨类= >“emojify🙊跨度< / > <代码>:speak_no_evil: < /代码> < / span > < / p > < br >

Emoji cheat sheet是一个有用的Emoji速记码参考。

＊＊＊

  • *注意:**以上步骤在Hugo中启用Unicode标准表情符号字符和序列，但是这些符号的呈现取决于浏览器和平台。要样式的表情符号，你可以使用第三方表情符号字体或字体堆栈;如。
